howToSteps (procedural, this is the core of the post):
1. Register the EMI scheme on HMRC's ERS Online Service (one-off) (HP13).
2. Gather the year's events: grants, exercises, lapses, adjustments, and the agreed valuations (HP13, HP14).
3. Complete the annual ERS return for the scheme (HP13).
4. If nothing happened, still file a NIL return (HP13).
5. Submit by 6 July following the tax-year end (HP13).
6. Keep the submission receipt and diarise next year's 6 July.
