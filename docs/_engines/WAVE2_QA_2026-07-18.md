## Defects found and fixed

### 1. Medical public tool fleet was 5/10 (claimed 10) — SEV: high
Session-6 built the 7 roster tools but registered tools 4-7 and 10 (nhs-superannuation-tiered-
contribution, nhs-pension-scheme-pays, gp-partner-drawings-planner, salaried-gp-vs-partner,
consultant-private-vs-nhs) in the PREMIUM registry only — blog-embedded, non-indexable. The
roster (`docs/medical/TOOL_ROSTER.md`) unambiguously specifies them as public.
**Fix:** 5 new public GenericTool configs + pure compute modules (reusing `nhs-super-tiers.ts`
single-source tiers) + golden tests, wired into `medical/web/src/lib/tools/registry.ts`.
Premium siblings retained. Built sitemap now emits all 10 `/calculators/<slug>` URLs (verified).

### 2. construction-cis income-tax band bug NOT fixed (claim said "fixed in 5 libs") — SEV: high
`cis-tax.ts saLiability()` had no PA taper above £100k and no 45% additional rate; three public
tools (cis-vs-paye-comparison, cis-self-assessment-calculator, cis-refund-estimator) inlined the
same flat math. The two NEW wave-2 tools had each inlined a *correct* taper as a workaround
instead of fixing the shared helper. Income tax was understated by £2,000+ for £110k-income cases.
**Fix:** correct `incomeTaxOn()` + `taperedPersonalAllowance()` added to `cis-tax.ts`;
`saLiability` and all three flat tools now route through it; one stale golden (pinned to the
buggy value, £18,289 → correct £20,289) re-derived and updated; new band-edge goldens added
(£60k/£120k/£150k). Stale "basic 20%, higher 40%" note copy updated to mention taper + 45%.
contractors-ir35 `tax2026.ts:109` was verified CORRECT as claimed. Property `landlordTax.ts`
remains FROZEN per owner instruction (not touched).

### 3. Medical `/tools/` links were 404s — SEV: medium
All related-links in medical tool configs (including the pre-existing salaried-doctor-take-home)
pointed to `/tools/<slug>`; the real route is `/calculators/<slug>` and no redirect exists.
Thank-you page linked `/tools` hub twice. **Fix:** all hrefs rewritten to `/calculators/`.
