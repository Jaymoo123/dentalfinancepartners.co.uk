## Site architecture: sub-segment hubs (/for/*)

One sector family, credible to each sub-segment (mirrors the rivals' own segmentation —
CareAxis, The Healthcare Accountant and SKN all split exactly this way):

```
/                                Care sector accountants (head family)
/for/care-homes                  residential + nursing; fee mix, occupancy, staffing %
/for/domiciliary-care            sleep-ins, travel time, hourly-rate economics — widest gap
/for/supported-living            rent/care split (VAT + housing benefit), only 1 dedicated rival
/for/childrens-homes             Ofsted-side sibling; rivals list it, nobody owns it
/for/care-startups               CQC registration finance, FVS, projections, start-up accounts
/calculators/*                   fleet per CALCULATORS.md (care-hour cost, sleep-in NMW, staffing margin first)
/research/care-provider-index    flagship asset (DATA_ASSET.md)
/blog/*                          clusters below
```

BUSINESS-audience rule honoured: every /for/* page carries the operator (owner/registered
manager) frame, segment-specific lead-form optional fields (beds/locations count, CQC
registration status, LA vs self-funder mix, sponsored-staff count), never the care-worker or
family-payer frame.

   metric; feeds internal links + GEO/AI-answer presence.

## Launch core (22 pages + 3 tools + 1 asset)

**Hubs (6):** homepage + the five /for/* pages above. Evidence: every dedicated rival
verified this run segments its offer this way; domiciliary and supported living are the two
segments where the dedicated tier is thinnest (2 firms each) while the query families are
broad (TOPICS.md clusters).

Measured per-hub demand (DataForSEO 2026-07-11; null = <10/mo):
- Homepage / head: "care home accountant(s)" 90/mo, KD 0, CPC £39.12; "accountants for
  care homes" 90/mo (carehome-accountants.co.uk's entire ranked surface is these 3 variants).
- /for/care-homes: head as above; supporting "care home vat" 30/mo KD 0.
- /for/domiciliary-care: "domiciliary care accountant" no measured volume; segment demand
  proxied by "cqc registration for domiciliary care" 90/mo KD 6 CPC £28.56 and the
  hourly-cost family (720/390/210 — see tools). Gap thesis unchanged: high commercial CPC,
  no dedicated ranker.
- /for/supported-living: "supported living accountant" no measured volume; "cqc supported
  living registration" 30/mo KD 43. Stays in the launch set on rival-thinness grounds, not
  volume.
- /for/childrens-homes: no measured volume on any variant — demoted to lowest hub
  priority; keep but write last (also pending the Ofsted-citation s6 pass).
- /for/care-startups: strongest measured hub — "cqc registration" 2,900/mo KD 26 CPC
  £10.80, "cqc registration manager" 720/mo KD 0, "cqc registration requirements" 260/mo
  KD 25, "cqc registration fees" 70/mo KD 0 CPC £13.07.

Measured priority order within hubs: **care-startups (CQC family) > care-homes/head >
domiciliary > supported-living > childrens-homes.**

**Money/service pages (6):**
- CQC financial viability statement service — a rival literally sells this (cqcfinancialviability.com); productised, fixed-fee.
- Care payroll (sleep-ins, travel time, sponsored staff) — 4 payroll bureaus have care pages; no dedicated firm owns the compliance angle end-to-end.
