# Calculator candidates — Hospitality (R3)

Date: 2026-07-11. Pattern: property calculator fleet (config-driven, 1 file + 1 import per
tool). Demand figures = DataForSEO Google Ads UK volumes pulled this run
(`raw/dfs_keyword_suggestions.json`, `raw/dfs_ranked_keywords.json`); precedent = verified
rival pages from `raw/verify_evidence.json` / sitemaps.

## Launch tier (build with the site)

1. **Tronc / tips PAYE & NIC calculator** — enter tips pool, distribution method, tronc vs
   direct-employer allocation → employee take-home + employer NIC saved (tronc via independent
   troncmaster is NIC-free per HMRC E24, citation verified live).
   Demand: "tronc" 2,900/mo KD0 CPC £12.99; "tronc meaning" 1,600; "what is tronc" 1,000;
   "tronc scheme" 170 CPC £18.98; "how to set up a tronc scheme" 50 CPC **£50.47**.
   Precedent: tipsandtroncs.co.uk runs a "Tronc Calculator"; WMT/Troncbox and Buzzacott
   monetise the same space — calculator is the wedge against them. 2024 Allocation of Tips Act
   + July-2026 "tipping changes" news (Moore Kingston Smith homepage) make it topical.
2. **Food & drink VAT rate checker** — item + hot/cold + eat-in/takeaway + delivery →
   0%/20% verdict with VAT Notice 709/1 + 701/14 logic.
   Demand: "vat on takeaway food" 390; "is there vat on takeaway food" 210; "do you pay vat on
   takeaway food" 110; hot/cold variants 70+70+40+20+20; "food vat"/"vat for food"/"vat on
   foodstuffs" 1,000 (ranked set). Clear DIY-informational cluster that captures the audience
   pre-hire; no rival has an interactive checker (verified sitemaps are articles only).
3. **Staff cost & rota margin calculator** — wage bill (NLW £12.71 from Apr 2026, verified
   live) + employer NIC 15% above £5,000 (ground truth; 2026-27 rates page verified) +
   pension + tronc treatment → true hourly staff cost, labour % of revenue vs 30% benchmark.
   Demand: indirect ("minimum wage hospitality" autocomplete family; labour-cost queries are
   long-tail) but this is the money-page tool: every operator does this monthly.

## Queue tier

4. **TOMS margin scheme estimator** — packaged trip revenue/direct costs → VAT under TOMS.
   Demand: "tour operators margin scheme" 210 + variants 60. Niche but zero interactive
   competition; HaysMac ranks with prose only.
5. **Business rates (retail, hospitality & leisure relief) checker** — RV + nation →
   relief % + estimated bill (2026 rates-cliff topicality; citation verified live).
6. **Gross profit / menu pricing calculator** — target GP% → menu price incl. VAT (wet/dry
   split). Trade-standard tool; Inn Control/Roslyns discuss GP in content but no tool.
7. **VAT registration threshold tracker** — rolling 12-month turnover vs £90,000 (verified
   live) — shared logic with estate calculators, cheap port.
8. **Flat Rate Scheme vs standard VAT comparator** — catering 12.5% FRS category (HMRC FRS7300
   manual verified live) vs standard; useful for cafes/takeaways under ~£150k.
